### langchain, langgraph 组织多 agent

### 多分析师(市场、社交媒体、新闻、基本面)、研究员(看涨、看跌)、管理者(研究经理、风险管理经理)、风险管理(激进分析师、保守分析师、中性分析师)，交易员

#### 第一阶段：数据收集与分析

1. **市场分析师** → 分析技术数据和价格趋势
2. **社交媒体分析师** → 分析市场情绪
3. **新闻分析师** → 分析最新新闻影响
4. **基本面分析师** → 分析公司基本面

#### 第二阶段：多空辩论

5. **多头研究员** → 提出看涨观点
6. **空头研究员** → 提出看跌观点
7. **循环辩论** → 两者交替进行，最多进行`max_debate_rounds`轮

#### 第三阶段：投资决策

8. **研究经理** → 综合所有分析，主持辩论，制定投资计划
9. **交易员** → 基于投资计划制定具体交易策略

#### 第四阶段：风险评估

10. **激进分析师** → 分析激进策略风险
11. **保守分析师** → 分析保守策略风险
12. **中性分析师** → 分析中性策略风险
13. **循环讨论** → 三者交替进行，最多进行`max_risk_discuss_rounds`轮
14. **风险管理经理** → 综合风险评估，做出最终交易决策
